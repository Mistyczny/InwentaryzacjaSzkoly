First run:
1. Zainstaluj nodejs w najnowszej wersji
2. restart kompa
3. W cmd windowswowym nakieruj path na ten folder
4. wpisz w cmd: npm install
5. po zakończeniu operacji wejdź w katalog ./src (jeden wyżej)
6. Tak jak w poprzednim wypadku wpisz npm install
7. Po zakończeniu instalacji wpisz w cmd ng serve aby skompilować projekt
8. Po zakończonej kompilacji otwórz przeglądarkę i wpisz: localhost:4200 - powinieneś ujrzeć rezultat.
INFO: ng serve również pełni funkcję watchera zmian w kodzie, jeżeli takie nastąpią automatycznie rekompiluje zmienione pliki i odświeża przeglądarkę
